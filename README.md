# Deadlands: Конструктор врагов

Статический MVP конструктора противников для Deadlands / SWADE.

Открыть: `index.html`.

Что уже есть:

- шаблоны врагов;
- параметры, навыки, атаки и особенности;
- автоматический расчет Защиты и Стойкости;
- заметки маршала;
- готовый статблок для копирования;
- локальное автосохранение;
- импорт и экспорт JSON;
- печатная версия статблока.
