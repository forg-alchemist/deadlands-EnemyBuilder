# Deadlands: Конструктор врагов

Статический MVP конструктора противников для Deadlands / SWADE.

Открыть: `Enemy-Builder.html`.

Что уже есть:

- шаблоны врагов;
- параметры, навыки, атаки и особенности;
- автоматический расчет Защиты и Стойкости по базовым формулам;
- каталоги Изъянов, Черт, Оружия и Брони из карты персонажа;
- загрузка арта врага с локальным сохранением;
- модификаторы Шага и Бега относительно базы 3 / d4;
- броня по секторам: каталоговая и ручные модификаторы головы, торса, рук и ног;
- заметки маршала;
- готовый статблок для копирования;
- локальное автосохранение;
- импорт и экспорт JSON;
- печатная версия статблока.
